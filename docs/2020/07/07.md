### 示例 - Vue3 SFC Improvements

```vue
<template>
  <button @click="inc">{{ count }}</button>
</template>

<script setup>
import { ref } from "vue";

export const count = ref(0);
export const inc = () => count.value++;
</script>
```

```vue
<template>
  <div class="text">hello</div>
</template>

<script>
export default {
  data() {
    return {
      color: "red"
    };
  }
};
</script>

<style :vars="{ color }">
.text {
  color: var(--color);
}
</style>
```
